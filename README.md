# App Internship Project

This workspace contains the initial structure for a mobile-first product built with:

- Frontend: React Native + Expo
- Backend: FastAPI + Python
- Database: Supabase
- Vector search / AI layer: ChromaDB

## Repository structure

- frontend/: React Native app shell
- backend/: FastAPI backend service
- docs/: product planning and Phase 0 materials

## Getting started

1. Open the frontend folder and install dependencies.
2. Open the backend folder and create a Python virtual environment.
3. Start shaping the MVP around one primary user journey.
