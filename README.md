# Career and Internship Management System

A phased project scaffold for a mobile-first career and internship management platform built with Expo, Supabase, FastAPI, and optional ChromaDB for intelligent search.

## Project structure

- apps/mobile - Expo React Native frontend
- apps/api - FastAPI backend service for AI and integration logic
- infra/supabase - database schema and setup assets
- docs - architecture and planning notes

## Phase 1 goals

- Create a clean starter project structure
- Set up frontend and backend folders
- Prepare for authentication, role-based access, internships, and applications

## Recommended stack

- Frontend: Expo + React Native + TypeScript
- Backend: Supabase for core data/auth/storage
- AI service: FastAPI
- Search enhancement: ChromaDB later
